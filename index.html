🤖 AI System Design

5 نماذج AI متكاملة لتجربة مخصصة بالكامل

Model 1 AI Recommendation Engine

Type: Hybrid (Content-based + Collaborative Filtering + LLM reranking)

Input: Quiz answers (15 questions) Input: Academic scores Input: Behavior data Output: Top 10 college matches + scores

Stack: Python FastAPI → Scikit-learn (initial matching) → OpenAI GPT-4o (reranking + Arabic explanation generation)

Target accuracy: 82%+ user satisfaction rate

Model 2 AI Chatbot — Arabic NLP

Architecture: RAG (Retrieval Augmented Generation) فوق قاعدة بيانات الكليات المصرية

LLM: GPT-4o / Gemini 1.5 Vector DB: pgvector (embeddings) Arabic NLP preprocessing Context: Student profile + chat history

Fallback: خط دعم بشري لو AI مش عارف يجاوب. Context window: آخر 20 رسالة. Streaming via Server-Sent Events.

Model 3 Salary Predictor

Type: Gradient Boosting Regressor (XGBoost) trained on Egyptian job market data

Features: college_type, specialization, governorate, years_exp Output: salary_min, salary_max, salary_median Data: LinkedIn + Forasna.com (quarterly update)

RMSE target: ±15% salary range accuracy

Model 4 Alternative Recommendations

Rule-based + ML: لو الطالب مش واصل للدرجة المطلوبة، بنحسب similarity score بين الكليات بناءً على subject areas، career outcomes، fee ranges ونقترح بدائل مناسبة بشرح سبب كل توصية.

Model 5 Behavior Tracking + Personalization

Real-time event tracking → Redis queue → async processing → تحديث user preference vector → إعادة ترتيب التوصيات. كل click وview وsave بيأثر على نتائج التوصيات في الوقت الفعلي.

Redis Streams Bull.js Queue pgvector embeddings Real-time updates
