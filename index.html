🤖 AI System Design

5 نماذج AI متكاملة

Model 1AI Recommendation Engine

Hybrid (Content-based + Collaborative Filtering + LLM reranking). Stack: Python FastAPI → Scikit-learn → GPT-4o reranking.

Target accuracy: 82%+ user satisfaction rate

Model 2AI Chatbot — Arabic NLP

RAG (Retrieval Augmented Generation) فوق قاعدة بيانات الكليات المصرية. LLM: GPT-4o. Vector DB: pgvector. Streaming via Server-Sent Events.

Model 3Salary Predictor

Gradient Boosting Regressor (XGBoost) trained on Egyptian job market data. Features: college_type, specialization, governorate, years_exp.

Model 4Alternative Recommendations

Rule-based + ML: similarity score بين الكليات بناءً على subject areas، career outcomes، fee ranges.

Model 5Behavior Tracking

Real-time event tracking → Redis queue → async processing → تحديث user preference vector → إعادة ترتيب التوصيات.
